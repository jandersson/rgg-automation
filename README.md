# rgg-automation

Tools to assist with the minigames in Ryu Ga Gotoku (RGG / "Yakuza") titles —
starting with **Judgment**. The aim is a *real-time advisor*: capture the
minigame on screen, read the cards, and suggest the mathematically best move
(probabilities, pot equity, basic strategy, card counting).

These are personal-use assistants for offline, single-player minigames.
